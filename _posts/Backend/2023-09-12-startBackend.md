# 앞으로 공부해야 할 것들

- graphql
- DB
- DBMS (mysql) -> 추천 받은거는 mysql workbench(이미 써봤지만 최신버전은 호환되지 않는다.) , datagrip

- nosql, sql 특징들
- docker 역할, 이미지, 컨테이너
- express
- sql 문 . select, use, where, join , order by, create 등등
