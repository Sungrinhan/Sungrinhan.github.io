파이썬 시작하기

## 프로그램설치: anaconda

- anaconda 란?

  - 파이썬 기본 컴파일러
  - 파이썬 주요 라이브러리
  - jupyter notebook 등 유용한 툴

- 참고
  - 컴파일러: 프로그래밍 언어로 작성된 코드를 컴퓨터가 실행할 수 있는 코드로 변환하는 프로그램(like babel)
  - 파이썬의 장점: **라이브러리**
    - pip install library-name

### Jupyter notebook 이란?

- Editor(PyCharm) VS jupyter notebook - 한줄 한줄 코드 실행 결과 확인이 쉽다. - 문서와 코드를 함께 작성/저장할 수 있다.
  > 복잡한 자료구조/ 알고리즘을 보다 쉽게 정리하고 익히기 위해 사용
